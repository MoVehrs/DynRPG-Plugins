BareHanded 1.0.1.0
DynRPG plugin for automatic unarmed weapon management in RPG Maker 2003. Enables automatic equipping of designated weapons when actors are unarmed, providing proper stats for barehanded combat. Supports per-actor configuration, variable-based weapon IDs, and dual-wielding compatibility.

◆ LoadConfig()

bool BareHandedConfig::LoadConfig ( char * pluginName)

Loads and validates configuration settings from DynRPG.ini.

Parameters
pluginNameName of the plugin section in DynRPG.ini.
Returns
True if configuration loaded successfully, false otherwise.

This function:

  • Clears existing configuration
  • Loads debug settings
  • Loads actor-to-weapon mappings
  • Validates weapon and variable IDs
    Note
    Both fixed weapon IDs and variable IDs must be positive values to be considered valid configurations.

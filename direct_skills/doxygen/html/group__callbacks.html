DirectSkills 1.0.1.1
DynRPG plugin for direct skill execution from battle commands in RPG Maker 2003. Allows battle commands to automatically execute designated skills instead of regular attacks. Features fixed and variable-based skill mappings with fallback options and built-in limit_break plugin compatibility.Allows battle commands to automatically execute designated skills instead of regular attacks.

◆ LoadConfig()

bool DirectSkillsConfig::LoadConfig ( char * pluginName)

Loads and validates configuration settings from DynRPG.ini.

Parameters
pluginNameName of the plugin section in DynRPG.ini.
Returns
True if configuration loaded successfully, false otherwise.

This function:

  • Clears existing configuration
  • Checks for conflicts with the limit_break plugin
  • Loads debug settings
  • Loads command-to-skill mappings
  • Validates variable-based mappings have default skill IDs
    Note
    Variable-based mappings require a corresponding default skill ID to handle cases where the variable contains an invalid value.
    Warning
    Command IDs that conflict with the limit_break plugin will be skipped to prevent interference between plugins.

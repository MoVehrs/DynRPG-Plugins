Detailed Description

Core implementation namespace for the DynamicQuickPatch plugin.

Handles dynamic memory patching and variable monitoring during gameplay. Provides functionality to automatically update memory values based on RPG Maker variable changes and game state.

◆ updateQuickPatch()

void DynamicQuickPatch::updateQuickPatch ( const DynamicQuickPatchConfig::QuickPatchMapping & mapping,
int value )

Updates a quickpatch mapping with a new value.

Parameters
mappingThe mapping definition to update.
valueThe new value to apply.

Applies the new value to memory according to the mapping type:

  • For 8-bit: Allows 0 as valid value, clamps to valid range
  • For 32-bit: Allows 0 as valid value, direct value write
  • For hex: Uses predefined hex string, 0 deactivates patch
    Note
    For hex patches, if value is 0, restores original memory values. For 8-bit and 32-bit patches, 0 is treated as a valid value to write.

DynamicQuickPatch 1.0.1.0
DynRPG plugin for dynamic battle parameter modification in RPG Maker 2003. Provides real-time modification of battle parameters through variables and switches. Enables dynamic adjustments to hit rates, critical rates, damage variance, and other combat calculations without database edits.

Detailed Description

Contains configuration settings and loading functionality.

Manages the loading and validation of plugin settings from DynRPG.ini, including variable-to-memory mappings and patch configurations.

enum DynamicQuickPatchConfig::QuickPatchType

Enumeration of supported quickpatch value types.

Defines the different types of memory values that can be written:

  • 8-bit signed integers (% notation)
  • 32-bit signed integers (# notation)
  • Raw hex values
